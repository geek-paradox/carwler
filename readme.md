# Carwler

### Prerequisites
Please install mongo db to use carwler

## Setting up project
#### Clone project
- `git clone git@github.com:rajat19/carwler.git`
- `python3 install -e carwler`
- `cd carwler`

#### Crawling Data and Saving to mongo
- `python3 carwler/crawler.py`

#### Start flask server
- `pyton3 api/main.py`
